# REST API Client (Python)

## 📌 Objective
This project demonstrates how to interact with a REST API using Python.

## 🚀 Features
- GET all posts
- GET post by ID
- POST (create post)
- PUT (update post)
- DELETE post
- CLI-based interface

## 🛠 Technologies Used
- Python
- requests library
